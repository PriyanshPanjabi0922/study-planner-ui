## Purpose

The goal of this project is to explore how a study planning interface can be presented clearly using only static web technologies.

It emphasizes:
- Clear weekly structure
- Visual status indication (Completed / Active / Pending / Upcoming)
- Distraction-free layout suitable for future expansion

## Features

- Hero section with conceptual illustration
- Weekly study overview with task status labels
- Product-style feature explanations
- Clean visual hierarchy and section separation
- Dark-themed, minimal UI focused on readability
